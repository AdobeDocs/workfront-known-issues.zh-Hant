# 整合：透過整合傳送至AEM時，檔案名稱為「null」

當透過Workfront整合將大型檔案（超過100 MB）傳送到Adobe Experience Manager時，AEM中的檔案名稱為「null」。

ZIP和TIF檔案已報告過此問題。

**解決方法**

執行下列任一項作業：

* 將檔名稱對應至AEM標題。
* 直接在AEM中輸入檔案名稱。

_首次回報於2024年4月23日。_

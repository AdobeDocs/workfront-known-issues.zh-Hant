# 整合：當透過整合傳送至 AEM 時，檔案名稱為「null」

當透過 Workfront 整合將大型檔案 (超過 100 MB) 傳送到 Adobe Experience Manager 時，AEM 中的檔案名稱為「null」。

ZIP 和 TIF 檔案都有回報此狀況。

**解決方法**

執行下列任一項作業：

* 將文件名稱對應到 AEM 標題。
* 直接在 AEM 中輸入檔案名稱。

_於 2024 年 4 月 23 日首次通報。_

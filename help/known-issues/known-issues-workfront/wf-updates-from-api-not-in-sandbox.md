# 更新：透過 API 發表的評論在沙箱環境中不可見

>[!NOTE]
>
>此問題已在2024年4月29日修正。

當透過 Workfront API 發表評論時，該評論在生產環境中可見。但該評論在沙箱環境中不可見。

_於 2024 年 4 月 8 日首次通報。_

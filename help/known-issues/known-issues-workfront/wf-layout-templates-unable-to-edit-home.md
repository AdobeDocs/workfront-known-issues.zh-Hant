# 配置範本：管理員無法編輯首頁配置範本

當管理員嘗試編輯使用者的首頁配置範本時，介面未回應且無法儲存變更。

**解決方法**

將無回應的範本指派給您自己，然後再進行編輯。

_於2024年10月10日首次通報。_

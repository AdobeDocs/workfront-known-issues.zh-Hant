# 檔案：檔案區域未顯示所有檔案

當使用者檢視物件的「檔案」區域時，並非所有附加至檔案的檔案都會顯示。

此問題僅適用於檔案區域的標準檢視。 「清單」檢視會如預期般運作。

**解決方法**

切換到檔案區域的清單檢視。

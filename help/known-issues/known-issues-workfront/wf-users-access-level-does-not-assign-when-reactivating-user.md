# 用戶：重新啟動使用者時未指派存取層級

當使用者重新啟用停用的使用者，並嘗試在 [!UICONTROL 重新啟用使用者] 窗口中，訪問級別下拉清單不填充為用戶類型，用戶無法選擇訪問級別。 如果用戶在訪問級別中鍵入並保存，則該訪問級別不會分配給重新激活的用戶。

**因應措施**

重新啟用使用者，然後在 [!UICONTROL 編輯人員] 窗口。

_首次回報於 2023 年 2 月 9 日。_

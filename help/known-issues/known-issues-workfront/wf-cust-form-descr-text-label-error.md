# 自訂表單：由於「描述性文字」欄位，無法編輯自訂表單

>[!NOTE]
>
> 此問題已於2022年8月11日修正。

當使用者嘗試編輯包含 [!UICONTROL 描述性文字] 欄位中，欄位的標籤不會填入。 使用者看見錯誤「[!UICONTROL 此欄位為必填欄位]&quot;下，使用者因此錯誤而無法編輯自訂表單。

_2022年7月14日首次報告。_

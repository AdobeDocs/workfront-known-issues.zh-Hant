# [!UICONTROL 資源管理]：無法變更日期 [!UICONTROL 使用率] 區域

當使用者檢視 [!UICONTROL 使用率] 區域 [!UICONTROL 資源管理] 和嘗試變更日期範圍，日期不會變更，且使用者會看到以下錯誤：

「[!UICONTROL 發生錯誤，我們正在努力解決這個問題。若要繼續您的工作，請嘗試重新整理此瀏覽器。]&quot;

_於2024年1月24日首次通報。_

# 整合：從Outlook建立請求時發生錯誤

當使用者嘗試從Workfront for Outlook整合建立請求，並新增附件時。 使用者看到「發生錯誤」訊息。 請稍後再試。」

已建立請求，且請求中有電子郵件附件的資料夾，但資料夾是空的，且附件未新增至請求。

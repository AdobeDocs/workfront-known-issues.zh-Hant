# 首頁：專案名稱顯示為「無存取權」

<!--valid issue, won't fix until legacy home is deprecated-->

當使用者在首頁中檢視「我的任務」小工具時，專案名稱欄位顯示為「無存取權」。 即便於「我的任務」小工具中可以看到專案名稱，也會發生這種情況。

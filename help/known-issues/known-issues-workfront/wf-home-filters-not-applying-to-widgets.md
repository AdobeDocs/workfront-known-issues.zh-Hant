# [!UICONTROL 首頁]：篩選器不適用於新增 [!UICONTROL 首頁] 體驗

當使用者將篩選器套用至新的 [!UICONTROL 首頁] 體驗，Widget會顯示篩選條件應排除的專案。

這已在自訂沙箱環境中報告。 預覽和生產環境篩選器中的相同Widget如預期般運作。

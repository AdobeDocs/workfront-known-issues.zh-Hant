# 請求：內嵌程式碼URL未正確載入請求佇列

>[!NOTE]
>
>此問題已在2025年4月9日修正。

當使用者使用內嵌程式碼URL前往請求佇列時，佇列會載入，但不會載入任何關聯欄位，且使用者無法填寫請求。 當佇列所屬的專案有多個相關聯的佇列時，就會發生這種情況。

**解決方法**

要求佇列的直接存取URL如預期般運作。

_於2025年3月18日首次通報。_

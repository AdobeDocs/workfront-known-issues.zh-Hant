# 任務：建立任務時未儲存計畫時數

>[!NOTE]
>
>此問題已在2025年12月3日修正。

當使用者建立任務並輸入規劃時數，然後儲存任務時，未儲存規劃時數。

**解決方法**

建立任務，然後編輯它以新增計畫時數。

_於2025年10月10日首次通報。_

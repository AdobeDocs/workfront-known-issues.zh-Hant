# 問題：對問題進行進階指派時發生錯誤

當使用者嘗試在Workfront中進行問題的進階指派時，該問題未被指派，並且使用者會看到以下錯誤訊息：

&quot;[!UICONTROL APIModel INTERNAL不支援欄位持續時間(OpTask)]&quot;

_首次回報於2024年6月28日。_

# 校訂：校訂仍保留在我的核准Widget上

>[!NOTE]
>
>此問題已在2024年3月28日修正。

應該會從我的核准Widget中消失的校訂仍保留在Widget上。 當多個使用者同時對校訂做出決定，或一個使用者做出決定並快速變更時，可能會發生這種情況。

_首次回報於2024年2月7日。_

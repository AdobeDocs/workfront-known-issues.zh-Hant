# 專案：由於「存取以刪除」錯誤而無法共用專案

嘗試共用專案時，使用者會收到錯誤：「您沒有足夠的存取許可權可刪除此專案」。 儘管使用者未嘗試刪除專案，且使用者身為專案所有者並擁有專案的共用和管理存取權，仍會發生這種情況。

_於2024年9月25日首次通報。_

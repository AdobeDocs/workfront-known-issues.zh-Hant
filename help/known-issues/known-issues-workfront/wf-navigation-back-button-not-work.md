# 導覽：「上一頁」按鈕未返回上一頁

當使用者在Workfront按一下他們的瀏覽器的返回按鈕，可能會出現以下情況之一。

* 瀏覽器標簽名稱會變更，但頁面不會變更。 再按一下返回按鈕即可解決問題。
* 系統會將使用者帶至其瀏覽器登陸頁面。

_首次回報於2024年6月3日。_

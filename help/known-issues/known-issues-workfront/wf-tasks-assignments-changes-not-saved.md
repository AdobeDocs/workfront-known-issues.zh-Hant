# 任務：在任務的編輯對話方塊中所做的編輯未儲存

>[!NOTE]
>
>* 此問題適用於舊版任務體驗。
>* 產品團隊目前正在評估此問題的解決方案，這可能需要產品增強功能。產品增強功能是在產品公告中而不是在維護更新中傳達。

當使用者在舊版任務對話方塊中對任務的「編輯」區段進行變更並儲存時，變更未儲存。

**解決方法**

對於某些變更，您可以使用新的任務體驗。

_於2025年11月24日首次通報。_

# 任務：編輯任務的工作分派區段未儲存

>[!NOTE]
>
>此問題適用於舊版任務體驗。

當使用者在舊版任務對話方塊中對任務的「編輯」區段進行變更並儲存時，變更未儲存。

**解決方法**

對於某些變更，您可以使用新的任務體驗。

_於2025年11月24日首次通報。_

# 版面配置範本：登陸頁面不正確

>[!NOTE]
>
>此問題已關閉，因為這不是問題。

當具有請求者授權的使用者被指派至將請求頁面設定為登陸頁面的版面配置範本時，該使用者將被導向至作為其登陸頁面的首頁。

已針對請求者授權報告此情況。

_於 2024 年 3 月 22 日首次通報。_

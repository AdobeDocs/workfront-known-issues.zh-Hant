# 版面配置範本：登入頁面不正確

當具有請求者授權的使用者被指派給將請求頁面設定為登陸頁面的版面配置範本時，該使用者被導向首頁作為其登陸頁面。

已針對請求者授權報告此問題。

_首次回報於2024年3月22日。_

# 設定：以其他使用者身份登入時，不會顯示「登入身份」橫幅。

當管理員以其他使用者身份登入時，未出現「登入身份」橫幅。 系統不會提醒使用者，他們是以其他使用者的身分登入，且無法登出該使用者的帳戶。

**解決方法**

重新整理該頁面。

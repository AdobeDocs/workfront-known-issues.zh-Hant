# 設定：以其他使用者身分登入時不會出現「登入身分」橫幅。

當管理員以其他使用者登入時，不會出現「登入身分」橫幅。系統不會提醒使用者他們以其他使用者身分登入，且無法登出該使用者的帳戶。

**解決方法**

重新整理該頁面。

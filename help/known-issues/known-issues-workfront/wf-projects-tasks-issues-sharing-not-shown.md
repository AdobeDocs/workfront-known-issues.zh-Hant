# 專案、任務和問題：未顯示繼承的許可權

>[!NOTE]
>
>此問題已在2024年4月22日修正。

當使用者共用物件時，顯示繼承許可權的卡片不存在。 這已在以下使用者報告：

* 物件的受指派人
* 正在輸入名稱以共用物件的使用者

_於2024年1月9日首次通報。_

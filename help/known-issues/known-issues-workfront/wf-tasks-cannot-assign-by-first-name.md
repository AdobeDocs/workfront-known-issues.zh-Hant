# 任務：在任務指派中搜尋名字未傳回使用者

當使用者進行任務指派並嘗試按名字搜尋受指派人時，搜尋未傳回任何使用者，且使用者看到「找不到結果」訊息。

**因應措施**

依全名搜尋會如預期傳回使用者。

_首次回報於2024年6月27日。_

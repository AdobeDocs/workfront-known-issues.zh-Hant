# 任務：在進行任務指派時搜尋名字不會傳回使用者

當使用者進行任務指派並嘗試按名字搜尋被指派者時，搜尋不會傳回任何使用者，使用者會看到訊息「找不到任何結果」。

**解決方法**

以全名搜尋會如預期傳回使用者。

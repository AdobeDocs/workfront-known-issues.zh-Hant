# 使用者：無法編輯其他使用者的通知設定

當使用者嘗試編輯另一個使用者的通知設定時，他們無法編輯設定，並且會看到以下訊息：

「您沒有足夠的存取許可權來編輯財務資料。」

_於2025年9月9日首次通報。_

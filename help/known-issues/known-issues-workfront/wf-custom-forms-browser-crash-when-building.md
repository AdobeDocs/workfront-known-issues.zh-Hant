# 自訂表單：建立大型表單時瀏覽器當機

當使用者在表單設計器中構建或編輯大型表單時，瀏覽器可能會當機，並且使用者無法建立或編輯表單。

這會影響表單設計工具。 這些表單可如預期用於物件。

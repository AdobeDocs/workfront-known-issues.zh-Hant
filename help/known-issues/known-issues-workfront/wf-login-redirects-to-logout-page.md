# 登入：登入頁面重新導向以登出登陸頁面

當使用者嘗試登入 [!DNL Workfront]，而並非登入頁面，而是導向他們登出時會著陸的頁面。

**因應措施**

輸入您的 [!DNL Workfront] URL並嘗試再次登入。

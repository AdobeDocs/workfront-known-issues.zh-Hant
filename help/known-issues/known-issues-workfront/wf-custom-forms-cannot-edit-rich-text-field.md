# 自訂表單：無法編輯RTF文字欄位

>[!NOTE]
>
>此問題已在2025年12月22日修正。

當使用者嘗試編輯自訂表單上的RTF文字欄位時，變更不會儲存，並且該欄位不會更新以反映變更。

_於2025年12月3日首次通報。_

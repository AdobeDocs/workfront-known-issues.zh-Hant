# 請求：複製請求時出現「您必須是系統管理員」錯誤

>[!NOTE]
>
>此問題已在2026年2月3日修正。

當使用者複製請求並嘗試儲存時，請求未儲存，且使用者看到以下錯誤訊息：

「您必須是系統管理員才能變更此自訂資料引數值。」

_於2025年1月12日首次通報。_

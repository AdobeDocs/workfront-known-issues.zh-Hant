# 校訂：關閉校訂會重新導向到檔案清單

當使用者檢視校訂並按一下X圖示將其關閉時，使用者將被重新導向至檔案清單而非檔案詳細資訊。

當下列兩項皆為真時，會發生此狀況：

* 使用者已從連結開啟校訂。 如果從檔案詳細資料開啟校訂，則在關閉校訂時將使用者導向回檔案詳細資料。
* 使用者屬於尚未移轉至Adobe Unified Shell的組織。

_於2025年5月12日首次通報。_

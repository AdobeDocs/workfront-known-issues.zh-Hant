# 工作負載平衡器：未正確計算每個使用者的時數

>[!NOTE]
>
>此問題已在2026年2月6日修正。

當使用者在工作負載平衡器，這些小時可能不正確地計算。 例如，周檢視可能會顯示一種小時計算，但4週檢視會顯示該周的不同小時數。

_於2026年1月27日首次通報。_

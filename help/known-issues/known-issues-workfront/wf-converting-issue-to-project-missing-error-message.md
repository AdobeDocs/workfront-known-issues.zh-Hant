# 問題：使用範本將問題轉換為專案時，系統會顯示錯誤訊息

>[!NOTE]
>
>此問題已於2022年9月1日修正。

使用範本將問題轉換為專案時，系統顯示含有「 」訊息的頁面[!UICONTROL 專案已不存在]」，而非說明轉換失敗原因的正確錯誤訊息。

**因應措施**

返回上一頁並再次選取範本會顯示適當的錯誤訊息，讓使用者能夠更正錯誤並重試轉換。

_2022年8月12日首次報導。_

# 自訂表單：描述性文字欄位顯示工具提示

>[!NOTE]
>
>此問題已於2022年9月13日修正。

當使用者檢視具有描述性文字欄位的自訂表單時，描述性文字欄位會顯示工具提示。 如果使用者暫留在工具提示上，工具提示會顯示與描述性文字欄位相同的文字。

_2022年8月17日首次報導。_

# 自訂表單：顯示邏輯在請求表單上無法正確顯示

當使用者填寫連線到請求的自訂表單時，顯示邏輯未遵循，並且某些欄位可能未按預期顯示。

**解決方法**

關閉請求草稿，然後重新開啟它。

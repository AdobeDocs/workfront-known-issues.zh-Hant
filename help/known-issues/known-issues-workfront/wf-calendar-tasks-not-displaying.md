# 日曆：未顯示任務

>[!NOTE]
>
>產品團隊目前正在評估此問題解決方案。 此問題解決後，會在產品公告中傳達，而不會透過維護更新。

當使用者有清單報表和具有相同篩選器的日曆報表時，他們會發現有些工作會顯示在清單報表中，但不會顯示在日曆報表上。

_2019年10月31日首次報導。_

# 報告：使用情況報告中的預算工時與透過 API 報告的預算工時不相符

>[!NOTE]
>
>此問題已在 2023 年 11 月 16 日修正。

當使用者對指定項目的 RPBGHR 物件進行 API 呼叫，並將該呼叫的結果與該專案的使用情況報告進行比較時，結果不相符。

_於 2023 年 9 月 20 日首次回報。_

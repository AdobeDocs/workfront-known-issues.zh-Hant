# 通知：存取電子郵件通知設定時發生錯誤

>[!NOTE]
>
>此問題已於2022年11月3日修正。

當使用者嘗試變更電子郵件通知設定時，他們可能會看到以下其中一個錯誤：

* 「[!UICONTROL 請再試一次。糟糕！發生錯誤。請聯絡 [!DNL Workfront]，以便我們找出問題並加以解決。]」

* 「[!UICONTROL 未能擷取電子郵件通知]」

這已在以下區域報告：

* [!UICONTROL 設定] > [!UICONTROL 電子郵件通知]
* [!UICONTROL 使用者]>[!UICONTROL 編輯使用者]
* [!UICONTROL 群組]

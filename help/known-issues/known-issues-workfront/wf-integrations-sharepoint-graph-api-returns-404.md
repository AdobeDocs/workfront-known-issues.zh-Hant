# 整合：404從 [!DNL SharePoint] [!UICONTROL （圖表API）]

>[!NOTE]
>
>此問題已關閉，因為它不是 [!DNL Workfront] 錯誤。

當用戶嘗試將文檔添加到 [!DNL Workfront] 從 [!DNL SharePoint] [!UICONTROL （圖表API）]，則不會新增檔案，且使用者會看見下列錯誤訊息：

&quot;[!UICONTROL 404未找到： [沒有屍體]]&quot;

_2022年8月29日首次報導。_

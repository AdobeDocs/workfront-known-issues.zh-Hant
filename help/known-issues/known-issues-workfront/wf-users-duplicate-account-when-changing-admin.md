# 使用者：變更系統管理員時建立的帳戶重複

當系統的主要管理員變更為其他使用者時，將會建立該使用者帳戶的重複專案，名稱為「管理員使用者」且電子郵件地址相同。

_於2024年12月6日首次通報。_

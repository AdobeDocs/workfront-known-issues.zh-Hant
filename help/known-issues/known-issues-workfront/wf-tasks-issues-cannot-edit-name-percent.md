# 任務和問題：無法編輯任務或問題的名稱或完成百分比

具有 [!UICONTROL Contribute] 對任務或問題的訪問無法編輯題頭中任務或問題的名稱。 此外， [!UICONTROL Contribute] 訪問無法編輯任務或問題的完成百分比。

_2022年10月6日首次報導。_

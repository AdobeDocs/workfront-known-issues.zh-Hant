# 權限：物件權限未正確繼承

>[!NOTE]
>
>產品團隊目前正在評估此問題的解決方案，這可能需要產品增強功能。 產品增強功能是在產品公告中而不是在維護更新中傳達。

繼承的權限未正確套用於物件。 這可能是由於繼承權限的複雜性所導致，而這些權限可能會受到以下因素的影響：

* 該物件與大量人員共享
* 大量物件受到繼承權限變更的影響

**解決方法**

限制物件的大小或複雜性可以幫助避免此問題。 我們建議任何父物件下的子物件數量不超過 10,000 個。

# 專案：使用範本建立的專案不符合範本的優先順序

>[!NOTE]
>
>此問題已在2024年5月7日解決。

當使用者使用範本將請求轉換為專案時，新建立的專案的優先順序與範本或請求不匹配。

_首次回報於2024年5月2日。_

# 看板：狀態變更時卡片未移動到適當的欄

當已連線卡片之連結物件的狀態在物件上直接變更時，卡片不會移動到適當的欄。 如果卡片上的物件狀態已變更，或卡片被拖曳到新欄，卡片會如預期般運作。

_於2023年11月28日首次通報。_

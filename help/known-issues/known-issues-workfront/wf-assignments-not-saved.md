# 指派：團隊指派未儲存

>[!NOTE]
>
>此問題已在2025年12月15日修正。

當使用者將團隊指派給Workfront中的工作專案時，未儲存團隊指派，並且團隊未收到工作通知。

_於2025年11月13日首次通報。_

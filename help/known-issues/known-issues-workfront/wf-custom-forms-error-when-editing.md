# 自訂表單：透過欄編輯自訂表單時發生錯誤

當使用者正在檢視包含自訂表單連結的物件清單，並嘗試透過按一下連結來編輯自訂表單時，使用者無法編輯表單，而是會看到以下訊息：

「已發生錯誤，而我們正在積極努力加以修正。」

**解決方法**

開啟已附加自訂表單的物件，並透過該物件的「詳細資訊」頁面編輯自訂表單。

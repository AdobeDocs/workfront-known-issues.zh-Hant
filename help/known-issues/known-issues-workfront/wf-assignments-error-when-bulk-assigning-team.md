# 分配：批量分配團隊時出錯

當用戶在分配個人後批量編輯任務或問題並分配團隊時，分配不會保存，並且用戶會看到以下錯誤：

&quot;[!UICONTROL 我們再試一次 — 發生以下錯誤：teamAssignments必須是對象清單或ID清單]&quot;

**因應措施**

先指派團隊，然後是個人。

# 優先順序：分組可防止排序和內聯編輯

>[!NOTE]
>
>產品團隊目前正在評估此問題的解決方案，這可能需要產品增強功能。產品增強功能是在產品公告中而不是在維護更新中傳達。

當「優先順序」工作清單已套用分組時，它無法排序，且內嵌編輯無法用於工作清單中的欄位。

_於2024年10月3日首次通報。_

# [!UICONTROL 資源規劃工具]：無法共用篩選器

>[!NOTE]
>
>此問題已在2024年6月10日修正。

當使用者嘗試分享篩選器在 [!UICONTROL 資源規劃工具]，他們無法共用篩選，且看到以下訊息：

&quot;[!UICONTROL 發生錯誤，我們正在努力解決變更。 若要繼續您的工作，請嘗試重新整理此瀏覽器頁面。]」

_首次回報於2024年5月31日。_

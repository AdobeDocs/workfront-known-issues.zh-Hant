# 校訂：從電子郵件連結存取校訂時缺少階層連結

當使用者按一下校訂通知電子郵件中的「前往校訂」按鈕時，缺少指向校訂父物件階層連結的連結。

**解決方法**

從 Workfront 中查看校訂，而非透過校訂通知電子郵件查看校訂。

# 報告：預設貨幣為美元時不載入報告

>[!NOTE]
>
>此問題已在2024年6月27日修正。

當使用者嘗試檢視預設貨幣為USD的報告時，該報告未載入。

**因應措施**

將預設貨幣變更為&quot;[!UICONTROL 專案的預設貨幣].」

_首次回報於2024年6月20日。_

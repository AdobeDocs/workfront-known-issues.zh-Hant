# [!UICONTROL 設定]：無法隱藏可選狀態

當使用者嘗試隱藏系統和群組層級的選用狀態時，狀態未隱藏。 如果使用者檢視狀態，隱藏狀態的選項未啟用，即使使用者確實啟用它並儲存變更。

# 工時單：&quot;[!UICONTROL 已存在計費小時數]&quot;在工時單上添加工時時出錯

>[!NOTE]
>
>此問題已於2022年8月25日修正。

當用戶嘗試在其時間表上向任務添加小時時，不會添加小時，用戶將看到以下錯誤消息：

[!UICONTROL （日期）此對象上已存在計費小時數。]

_2022年7月22日首次報告。_

# [!UICONTROL 首頁]：舊版中的搜尋功能無法運作 [!UICONTROL 首頁工作清單]

>[!NOTE]
>
>這個問題已解決。

當使用者檢視他們的 [!UICONTROL 首頁工作清單] 在舊版「首頁」體驗中並嘗試搜尋清單時，搜尋無效。

_於 2023 年 8 月 18 日首次通報。_

# 專案：如果已附加Web校訂，則無法將問題轉換為專案

當問題具有附加的網頁校訂(使用SharePoint等外部檔案提供者的連結的URL校訂)，並且使用者嘗試將該問題轉換為專案時，轉換失敗並且未建立專案。 使用者看到以下錯誤：

「複製檔案（檔案GUID）時發生問題。 請移除檔案或連絡支援，然後再試一次。」

_於 2024 年 3 月 4 日首次通報。_

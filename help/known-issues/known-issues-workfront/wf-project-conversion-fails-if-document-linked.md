# 專案：如果附加網頁校訂，則無法將問題轉換為專案

>[!NOTE]
>
>此問題已在2024年4月18日修正。

當問題有附加網頁校訂 (使用來自外部文件提供者 (例如 SharePoint) 之連結的 URL 校訂)，並且使用者嘗試將該問題轉換為專案時，轉換會失敗並且不會建立專案。使用者會看到以下錯誤：

「複製檔案 (檔案 GUID) 時出現問題。請刪除該檔案或聯絡支援人員並重試。」

_於 2024 年 3 月 4 日首次通報。_

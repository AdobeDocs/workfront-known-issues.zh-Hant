# 專案：無法新增問題至已完成的專案

>[!NOTE]
>
>此問題已在2024年10月24日修正。

當使用者嘗試將問題新增到完成狀態的專案時，使用者無法新增問題。 即使「新增和編輯問題」選項已啟用，也可能會發生這種情況。

_於2024年10月14日首次通報。_

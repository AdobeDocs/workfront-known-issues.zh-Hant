# 任務：具有子系之父系任務的經過天數期間不正確

當父系任務具有包含在「經過天數」計算中的子系任務時，父系任務的「經過天數」會錯誤地小於子系任務的合併「經過天數」。

_於2024年10月22日首次通報。_

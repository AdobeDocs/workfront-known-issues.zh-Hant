# [!UICONTROL  首頁]：使用 [!DNL Chrome] 以外的瀏覽器時出現的[!UICONTROL  首頁]摘要問題

>[!NOTE]
>
>由於無法重現，此問題已關閉。


當使用者查看他們的[!UICONTROL 首頁]並使用 [!DNL Chrome] 以外的瀏覽器時，他們可能會看到以下問題：

* 使用者不能編輯欄位，也不能向空欄位新增值。
* 即使顯示的工作物件上不存在欄位，也可能會顯示欄位。

這些問題已在以下瀏覽器中報告：

* [!DNL Safari]
* [!DNL Firefox]
* [!DNL Microsoft Edge]

_於 2021 年 11 月 23 日首次通報。_

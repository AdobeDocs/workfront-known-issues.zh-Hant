# 通知：設定通知偏好設定時出現「糟糕」錯誤

當使用者嘗試檢視通知偏好設定時，他們會看到以下錯誤：

「[!UICONTROL 糟糕！發生錯誤。請聯絡 Workfront，以便我們找出問題並加以修正。]」

以下區域已回報此問題：

* 使用者設定檔上的通知設定
* 設定中的事件通知區域

_首次回報於2024年7月31日。_

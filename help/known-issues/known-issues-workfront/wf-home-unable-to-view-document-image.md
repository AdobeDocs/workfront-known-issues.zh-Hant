# [!UICONTROL 首頁]:無法在 [!UICONTROL 工作清單]

>[!NOTE]
>
>產品團隊目前正在評估此問題解決方案。 此問題解決後，會在產品公告中傳達，而不會透過維護更新。

使用者點按 [!UICONTROL .docx] 縮略圖可查看文檔的預覽，視頻播放器將開啟，他們看不到文檔。

**因應措施**

按一下檔案名稱連結，直接前往 [!UICONTROL 文檔詳細資訊] 預覽文檔的頁面。

_2020年10月26日首次報導。_

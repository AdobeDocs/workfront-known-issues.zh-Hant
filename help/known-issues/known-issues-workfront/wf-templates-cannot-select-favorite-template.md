# 範本：無法從中選取範本 [!UICONTROL 我的最愛] 清單

當使用者嘗試從 [!UICONTROL 我的最愛] 清單中，當使用者將滑鼠移到清單中時，清單會消失，且使用者無法選取範本。

**因應措施**

執行下列任一項作業：

* 將滑鼠移回 [!UICONTROL 我的最愛範本] 選項，然後再次嘗試將滑鼠移到清單上。 這可能需要多次嘗試。
* 使用方向鍵選取範本。

_於 2023 年 11 月 14 日首次回報。_

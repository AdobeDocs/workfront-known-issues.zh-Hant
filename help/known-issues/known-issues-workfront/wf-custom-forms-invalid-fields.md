# 自訂表單：無法新增「estimateByHours」或「hoursPerPoint」工作列位

當使用者嘗試將欄位「estimateByHours」或「hoursPerPoint」新增到表單產生器中的自訂表單時，顯示錯誤訊息：「此自訂運算式無效，請再試一次。」

_於2024年10月9日首次通報。_

# 更新：回覆其他公司的使用者時畫面變成空白

當使用者嘗試回覆另一個公司的使用者評論時，畫面變為空白。

這是因為使用者沒有檢視其他公司使用者的許可權。

_於2024年1月8日首次通報。_

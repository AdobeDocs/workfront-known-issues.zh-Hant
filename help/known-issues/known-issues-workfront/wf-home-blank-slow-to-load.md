# [!UICONTROL 首頁]： [!UICONTROL 首頁工作清單] 空白/載入緩慢

當使用者嘗試檢視他們的 [!UICONTROL 首頁工作清單]，頁面載入時會長時間留白。 載入可能需要幾分鐘的時間，在此期間 [!UICONTROL 工作清單] 為空白。

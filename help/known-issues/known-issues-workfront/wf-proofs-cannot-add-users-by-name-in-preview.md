# 校訂：無法在預覽環境中依名稱將使用者新增到校訂中

>[!NOTE]
>
>此問題已在預覽沙箱環境中報告

當使用者嘗試將另一個使用者新增到校訂中，並按名稱搜尋該使用者時，名稱未出現，使用者無法選擇它以將另一個使用者新增到校訂中。

即使重新整理預覽沙箱環境，也可能發生這種情況。

_首次回報於2025年5月5日。_

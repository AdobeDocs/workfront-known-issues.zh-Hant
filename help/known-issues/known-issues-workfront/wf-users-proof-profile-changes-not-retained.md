# 使用者：校訂設定檔的變更問題

>[!NOTE]
>
>此問題已在2025年12月17日修正。

已報告有關變更使用者校訂設定檔的以下問題：

* 當使用者將不同的校訂設定檔指派給使用者並儲存時，未儲存變更，並且使用者保留以前的校訂設定檔。
* 即使已指派正確的校訂設定檔，使用者的校訂設定檔仍會顯示為不正確的校訂設定檔。
* 對使用者進行其他變更時，其「校訂」許可權也可能會變更。

_於2025年11月24日首次通報。_

# 專案：無法將專案移動到指定的計畫

>[!NOTE]
>
>此問題已在2025年7月8日修正。

當使用者嘗試將專案移動到計畫時，專案未移動。 即使使用者收到訊息說專案已成功移動，也可能發生這種情況。

_於2025年6月10日首次通報。_

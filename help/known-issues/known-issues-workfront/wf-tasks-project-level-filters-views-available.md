# 任務：可用於任務的專案層級篩選和檢視

>[!NOTE]
>
>此問題已在2025年2月21日修正。

開啟任務的篩選器或檢視清單時，在專案層級建立的篩選和檢視會出現在選項清單中。

_於 2025 年 1 月 27 日首次通報。_

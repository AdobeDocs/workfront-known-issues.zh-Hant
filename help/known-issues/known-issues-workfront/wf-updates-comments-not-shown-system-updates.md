# 更新：附加在系統更新的評論未顯示

<!--

-->

>[!NOTE]
>
>此問題已關閉，因為它已如期工作。

當使用者對系統動作 (例如狀態變更) 發表評論時，該評論不會顯示在「更新」區域中。

新的評論體驗有回報此問題。

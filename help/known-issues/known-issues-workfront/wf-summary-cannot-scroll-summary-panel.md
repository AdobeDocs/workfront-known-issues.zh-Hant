# [!UICONTROL 摘要]:無法捲動 [!UICONTROL 摘要] 面板

當用戶查看對象的 [!UICONTROL 摘要] 面板和嘗試捲動時，面板不會捲動，且使用者無法進一步向下檢視項目 [!UICONTROL 摘要] 中。 如果使用者按一下 [!UICONTROL 摘要] 面板即可跳至該區域，整個頁面會捲動而停止回應。 用戶必須刷新頁面才能返回對象清單。

_於 2023 年 1 月 26 日首次通報。_

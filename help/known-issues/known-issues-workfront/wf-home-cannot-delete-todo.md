# 首頁：無法刪除 [!UICONTROL 待辦事項] 新增專案 [!UICONTROL 首頁] 體驗

當使用者在新 [!UICONTROL 首頁] 體驗嘗試從刪除專案 [!UICONTROL 待辦事項] widget，專案不會刪除，且使用者會看到下列錯誤：

&quot;[!UICONTROL 移除您的待辦事項時發生問題，請稍後再試。]&quot;

當有小時記錄到時，可能會發生這種情況 [!UICONTROL 待辦事項] 個專案。

_於 2023 年 7 月 28 日首次通報。_

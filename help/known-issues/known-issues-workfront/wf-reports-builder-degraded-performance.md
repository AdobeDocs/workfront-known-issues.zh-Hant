# 報表：在Report Builder中建立和編輯報表時發生的問題

在Report Builder中建立和編輯報表時，系統回報了下列問題：

* Report Builder不會載入。
* 圖表分組（軸）消失。
* 此 [!UICONTROL 儲存+關閉] 按鈕沒有響應。

**因應措施**

重新載入頁面可讓您建立或編輯報表。

_2022年9月2日首次報導。_

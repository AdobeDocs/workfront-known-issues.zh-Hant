# 請求：無法存取請求頁面上的儀表板(「……/submitted」 URL)

>[!NOTE]
>
>此問題已在2024年5月16日修正。

嘗試存取請求頁面上的儀表板時，URL會暫時顯示儀表板URL，然後快速切換到已提交的URL (/submitted)。

_首次回報於2024年5月16日。_

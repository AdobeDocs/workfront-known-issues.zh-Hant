# 通知：評論中提及時，使用者不會收到電子郵件通知

>[!NOTE]
>
>此問題已結束，因為無法重現。

使用@提及格式在留言中提及使用者時，使用者不會收到電子郵件通知他們留言。 即使當有人將其納入直接更新時，使用者已啟用電子郵件通知的設定，仍可能會發生此狀況。

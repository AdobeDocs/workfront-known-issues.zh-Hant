# 任務：重新計算時間表的問題

已報告有關時間表重新計算的以下問題：

* 更新時間表任務的工期時，系統需要相當長的時間來重新計算。
* 當重新計算完成時，日期可能會維持停用或變成停用，就像時間軸仍在重新計算一樣。

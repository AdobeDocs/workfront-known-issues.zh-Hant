# [!UICONTROL 首頁]：專案未出現在 [!UICONTROL 首頁工作清單]

當使用者檢視他們的 [!UICONTROL 首頁工作清單]，應出現在以下位置的專案： [!UICONTROL 工作清單] 不存在。 例如，使用者可能會前往專案，檢視他們是否在專案中被指派任務，但這些任務未出現在使用者的任務中 [!UICONTROL 首頁工作清單].

_於 2023 年 5 月 31 日首次通報。_

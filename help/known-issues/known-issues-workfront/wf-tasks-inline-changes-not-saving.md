# 任務：對任務的內聯編輯未儲存

>[!NOTE]
>
>此問題適用於舊版任務體驗。

當使用者對舊版任務體驗中的任務進行內聯變更並儲存時，變更未儲存。

**解決方法**

對於某些變更，您可以使用新的任務體驗。

_於2025年12月4日首次通報。_

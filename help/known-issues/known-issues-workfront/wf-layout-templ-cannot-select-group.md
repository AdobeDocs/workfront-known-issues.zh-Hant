# 版面範本：建立佈局模板時無法選擇組

>[!NOTE]
>
>產品團隊目前正在評估此問題解決方案，這可能需要產品增強功能。 產品增強功能會在產品公告中傳達，而不會在維護更新中傳達。

身為群組管理員的使用者嘗試在 [!UICONTROL 介面] 區域 [!UICONTROL 設定]，系統會提示使用者選取 [!UICONTROL 管理存取] 欄位。 使用者輸入群組名稱，但群組未出現在下拉式清單中，且無法選取。 因此，使用者無法儲存版面範本。

**因應措施**

從群組的頁面(位於 [!UICONTROL 群組] 區域。

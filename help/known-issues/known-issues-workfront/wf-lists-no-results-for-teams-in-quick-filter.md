# 清單： [!UICONTROL 快速篩選] 搜尋團隊時不傳回任何結果

當使用者嘗試使用 [!UICONTROL 快速篩選] 在要搜索團隊的清單上，輸入團隊的名稱不會返回任何結果，即使該團隊在清單中可見(例如 [!UICONTROL 指派給] 欄位)。 搜索單詞&quot;[!UICONTROL 團隊]&quot;也不會傳回任何結果。

_2022年10月6日首次報導。_

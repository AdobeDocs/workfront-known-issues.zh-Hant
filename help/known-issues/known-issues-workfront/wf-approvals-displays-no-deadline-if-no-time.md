# 核准：當時間未包含在截止日期中時，核准會顯示為「無截止日期」

當核准截止日期僅包含日期而不是時間時，它會在核准首頁Widget中顯示為「無截止日期」的截止日期。

**解決方法**

在截止日期中包含時間。

# 項目、任務和問題：已調整大小 [!UICONTROL 說明] 框返回到原始大小

>[!NOTE]
>
> 此問題已於2022年9月9日修正。

當使用者重新調整 [!UICONTROL 說明] 框 [!UICONTROL 詳細資訊] 工作項的區域，使其變大，然後開始鍵入框，框將返回到其原始大小。 使用者仍可在方塊中輸入內容，內容會如預期般儲存。

_2022年8月18日首次報導。_

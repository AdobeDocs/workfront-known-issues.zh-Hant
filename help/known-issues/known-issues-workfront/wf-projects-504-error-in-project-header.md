# 專案：專案標題中的 504 錯誤

>[!NOTE]
>
>這個問題已解決。

當使用者檢視專案時，專案標題中沒有名稱或資料。 使用者反而會看到類似於以下的錯誤：

「504：未定義 /internal/qs/customizableHeaderData?objCode=PROJ&amp;objectID=xxxxxxxxxxxxxxx」

_於 2023 年 2 月 3 日首次通報。_

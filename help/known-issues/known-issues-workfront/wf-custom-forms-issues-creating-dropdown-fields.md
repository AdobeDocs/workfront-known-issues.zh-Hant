# 自訂表單：自訂下拉式清單欄位中的問題

以多選欄位啟用的自訂下拉式欄位可能會顯示下列問題：

* 「+」[!UICONTROL 新增]「 」按鈕不在編輯模式下。
* 沒有值的欄位會顯示「 — [!UICONTROL 無標籤] — 」選項。

**因應措施**

嘗試以下任一作業：

* 移除要多選欄位的選項。
* 設定預設值。

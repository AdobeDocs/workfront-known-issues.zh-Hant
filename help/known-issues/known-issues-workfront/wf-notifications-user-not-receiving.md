# 通知：使用者未收到預期的通知

當發生應傳送通知給使用者的事件時，該使用者未收到通知。 即使使用者通知設定允許通知，也可能會發生這種情況。

此問題會影響應用程式內通知和電子郵件通知。

_於2026年2月12日首次通報。_

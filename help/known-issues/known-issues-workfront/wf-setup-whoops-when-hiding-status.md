# 狀態： [!UICONTROL 糟糕] 隱藏狀態時發生錯誤

當使用者嘗試隱藏狀態時，狀態不會隱藏，並且使用者會看到以下錯誤：

「[!UICONTROL 糟糕！發生錯誤。請聯絡 Workfront，以便我們找出問題並加以修正。]&quot;
_於2023年11月30日首次通報。_

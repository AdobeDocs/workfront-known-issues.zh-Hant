# 請求：新請求體驗中的草稿問題

>[!NOTE]
>
>此問題已在2026年1月23日修正。

已報告有關新請求體驗中草稿的以下問題：

* 建立為草稿然後提交的請求只會顯示「請求」清單中主題的第一個字母。
* 捨棄的草稿不會從請求清單中移除，並會導致「無法擷取草稿」錯誤。

_於2025年12月30日首次通報。_

# 請求：「全部」檢視未顯示所有請求

>[!NOTE]
>
>* 此問題已在新請求體驗中報告。
>* 此問題已在2026年2月18日修正。

當使用者在新請求體驗中檢視請求清單，並且檢視設定為顯示所有請求，僅顯示部分請求。

_於2025年12月16日首次通報。_

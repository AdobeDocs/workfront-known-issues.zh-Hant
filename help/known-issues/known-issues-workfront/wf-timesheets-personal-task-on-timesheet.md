# 時程表：個人任務出現在時程表上

當使用者建立任務於 [!UICONTROL 待辦事項] 新增的Widget [!UICONTROL 首頁] 體驗，該任務會出現在使用者的時程表上。 即使任務沒有記錄時數，並且個人專案未固定，也會發生這種情況。

_於 2023 年 10 月 3 日首次通報。_

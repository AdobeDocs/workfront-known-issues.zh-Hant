# 整合： [!DNL SharePoint] 資料夾所有者未顯示資料夾

>[!NOTE]
>
>此問題已關閉，因為它不是 [!DNL Workfront] 錯誤。

當使用者嘗試在 [!DNL SharePoint] [!UICONTROL （圖表API）] 整合時，資料夾不會顯示。 即使使用者是資料夾的擁有者，也會發生此情況。

_2022年8月19日首次報導。_

# 校樣：關閉校樣時，使用者沒有返回上一頁

<!--This is on the WF page as well as the WFP page-->

在內檢視校樣的使用者 [!DNL Workfront] 關閉校樣，他們不會在開啟校樣前傳回所在的頁面。 而是會被重新導向至 [!DNL Workfront].

_2022年9月15日首次報導。_

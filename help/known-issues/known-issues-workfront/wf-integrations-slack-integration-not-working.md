# 整合：Workfront與Slack整合的問題

已報告有關用於Slack整合的Workfront的以下問題：

* 在Workfront中進行更新時，使用者不會在Slack中收到通知。
* 使用者無法從Slack登入Workfront。
* 用於Slack整合的Workfront中的其他命令無法如預期運作。

_於2025年6月30日首次通報。_

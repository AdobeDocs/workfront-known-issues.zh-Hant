# 自訂Forms:將問題轉換為專案時，不會保留依區段分隔隱藏之欄位中的資料

>[!NOTE]
>
> 此問題已於2022年8月11日修正。

當使用者將問題轉換為專案，且問題包含自訂表單，且區段分隔中含有資料，且可使用顯示邏輯隱藏時，該區段中的資料不會帶入新專案。

_2022年7月20日首次報告。_

# 使用者：大量更新使用者的速度非常慢

>[!NOTE]
>
>僅沙箱環境已報告此問題。

當使用者嘗試大量編輯使用者時，編輯視窗需要很長時間才能開啟。 即使只有兩名使用者也是如此，但更多使用者需要更多時間。

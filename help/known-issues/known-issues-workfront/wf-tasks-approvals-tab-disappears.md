# 任務：核准區域從左側導覽中消失

>[!NOTE]
>
>此問題已在2025年8月1日修正。

當使用者檢視任務時，核准區域可能從左側導覽中消失。 移除並新增核准區域到版面配置範本解決了問題，但問題可能會再次發生。

_於2025年6月18日首次通報。_

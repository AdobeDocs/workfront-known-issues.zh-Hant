# 任務：前置任務工具提示未顯示任務名稱

>[!NOTE]
>
>此問題已在2024年3月28日修正。

當使用者檢視任務清單並將滑鼠懸停在前置任務圖示上以取得更多資訊時，出現的工具提示不顯示前置任務的名稱。

_首次回報於2024年3月5日。_

# Workfront：整個應用程式的日期格式變更

<!--
>[!NOTE]
>
>This issue was fixed on August 30, 2024.
-->

在應用程式中具有清單的不同頁面之間導覽時，日期格式偶爾會從MM/DD/YYYY變更為DD/MM/YYYY，反之亦然。

# [!UICONTROL 首頁]： [!UICONTROL 請求] Widget不會顯示所有指派的團隊

當使用者檢視 [!UICONTROL 請求] 新增Widget [!UICONTROL 首頁] 體驗，即使請求實際指派了多個團隊，請求仍會顯示為僅指派一個團隊。

_於 2023 年 8 月 25 日首次通報。_

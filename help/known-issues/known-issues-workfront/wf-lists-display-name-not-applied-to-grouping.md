# 清單：顯示名稱未應用於自定義欄位分組

>[!NOTE]
>
>產品團隊目前正在評估此問題解決方案。 此問題解決後，會在產品公告中傳達，而不會透過維護更新。

當使用者依自訂欄位將清單分組，且變更顯示名稱時， [!UICONTROL 文字模式]，則會改為顯示欄位的原始名稱。

_2020年8月13日首次報導。_

# 任務：重新計算時間表時，任務日期因延長時間而變灰

當使用者在觸發重新計算時間表的工作清單中進行變更時，也會重新計算受該時間表影響的其他任務。 發生此情況時，這些其他任務中受影響的日期會變灰。 在此問題中，任務會長時間保持灰色狀態。

當這些日期變成灰色時，針對這些變成灰色日期所做的變更會予以儲存。

**因應措施**

重新整理該頁面。

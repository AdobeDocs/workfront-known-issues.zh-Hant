# [!UICONTROL 設定]:清單中載入緩慢 [!UICONTROL 設定]

>[!NOTE]
>
>產品團隊目前正在評估此問題的解決方案，這可能需要產品增強功能。產品增強功能是在產品公告中而不是在維護更新中傳達。

當使用者位於 [!UICONTROL 設定] 區域並嘗試載入清單，則清單載入的速度會非常緩慢。 這一情況已在各地的不同地區得到報導 [!UICONTROL 設定].

_首次回報於 2022 年 10 月 20 日。_

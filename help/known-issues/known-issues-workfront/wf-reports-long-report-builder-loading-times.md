# 報告：報告載入時間過長

>[!NOTE]
>產品團隊正在透過產品增強功能解決此問題。產品增強功能會在產品公告中宣布，而非透過維護更新宣布。

報告需要長時間才能載入。這樣會影響檢視報告，以及在報告產生器中編輯和建立報告。

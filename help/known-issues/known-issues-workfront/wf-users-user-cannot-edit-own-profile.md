# 用戶：用戶無法編輯自己的用戶設定

若使用者擁有 [!UICONTROL 工作], [!UICONTROL 檢閱]，或 [!UICONTROL 要求] 嘗試編輯自己的用戶設定，開啟的彈出窗口為空，用戶無法進行任何編輯。 若要退出快顯視窗，使用者必須重新整理頁面。

**因應措施**

使用 [!UICONTROL 計畫] 授權可對使用者進行變更。

_於 2023 年 1 月 18 日首次通報。_

# 更新：編輯評論時，可以使用已停用的影像選項

之後 [!DNL Workfront] 管理員已停用將影像新增至註解的選項，建立註解時無法使用該選項。 但是，如果使用者編輯現有的註解，則影像選項可用。

_於2024年1月29日首次通報。_

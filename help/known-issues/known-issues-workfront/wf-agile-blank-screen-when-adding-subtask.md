# 敏捷：新增子任務時故事板變成空白

當使用者嘗試在選取篩選器時將子任務新增到故事板時，畫面變為空白，使用者無法新增子任務。

**解決方法**

重新整理頁面、取消選取篩選器，然後新增子工作。

_首次回報於2024年7月8日。_

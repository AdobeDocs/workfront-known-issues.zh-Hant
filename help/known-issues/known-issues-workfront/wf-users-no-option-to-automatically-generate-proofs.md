# 使用者：使用者設定檔中缺少自動產生校訂的選項

當使用者檢視使用者設定檔時，「上傳檔案時自動產生校樣」選項無法使用。 即使滿足選項的所有先決條件，仍可能發生這種情況。

_於2025年12月5日首次通報。_

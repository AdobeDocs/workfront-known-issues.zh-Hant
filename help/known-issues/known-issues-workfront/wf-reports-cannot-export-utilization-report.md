# 報表：無法導出利用率報告

>[!NOTE]
>
>此問題已於2022年8月4日修正。

當用戶嘗試導出利用率報告時，這些報告會被導向到空白頁面，而報告從不導出。 嘗試從匯出報表時，已回報此問題 [!UICONTROL 資源] 頁面和在專案內匯出時。

_2022年7月26日首次報告。_

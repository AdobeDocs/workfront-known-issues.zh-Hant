# 資源規劃工具：「匯出」按鈕已停用

>[!NOTE]
>
>此問題已在2025年6月9日修正。

當使用者嘗試從資源規劃工具匯出時，按鈕被停用（灰顯）並且使用者無法匯出。

_首次回報於2025年5月20日。_

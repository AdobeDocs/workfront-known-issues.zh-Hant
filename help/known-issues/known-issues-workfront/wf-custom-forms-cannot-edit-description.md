# 自訂表單：編輯描述性文字欄位時發生錯誤

當使用者嘗試編輯自訂表單上的描述性文字時，該文字未儲存，並且使用者會看到以下錯誤：

「重複的索引鍵值違反唯一條件約束」

這已在舊版表單產生器中報告。

**解決方法**

使用新的表單設計工具Beta版來編輯表單。

_首次回報於2024年5月23日。_

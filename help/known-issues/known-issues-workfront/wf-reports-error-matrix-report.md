# 報告： 」[!UICONTROL 糟糕]「檢視矩陣報告時發生錯誤

當使用者檢視矩陣報告時，他們會看到以下錯誤：

「[!UICONTROL 糟糕！發生錯誤。請聯絡 Workfront，以便我們找出問題並加以修正。]」

當報告依日期和&quot;[!UICONTROL 顯示沒有結果的周]「 」選項已啟用。

**因應措施**

停用&quot;[!UICONTROL 顯示沒有結果的周]」選項。

# 任務：建立新任務時視窗關閉

當使用者嘗試使用「+新增任務」按鈕建立新任務時，「新增任務」視窗會開啟，但在使用者輸入任何資訊之前會關閉。 因此，使用者無法使用此方法建立新工作。

_於2023年12月18日首次通報。_

# 請求：無法共用請求佇列路徑

>[!NOTE]
>
>此問題已在2024年4月18日修正。

當使用者嘗試共用請求佇列路徑時，連結未開啟正確的請求佇列。 相反地，連結會前往請求頁面。

_首次回報於2024年4月18日。_

# [!UICONTROL 首頁]: [!UICONTROL 專案名稱] 在 [!UICONTROL 工作清單]

>[!NOTE]
>
>產品團隊目前正在評估此問題解決方案。 此問題解決後，會在產品公告中傳達，而不會透過維護更新。

當要求使用者核准無法存取的證明時，使用者和使用者 [!UICONTROL 工作清單] in [!UICONTROL 首頁] 排序依據 [!UICONTROL 專案名稱]，使用者無法看見專案名稱，而會改為顯示GUID。

# 專案：共用專案時發生錯誤

>[!NOTE]
>
>此問題已在2024年5月2日修正。

當使用者嘗試共用專案時，專案不會共用，並且使用者會看到以下訊息：

「您沒有足夠的存取許可權來刪除此專案」

即使使用者嘗試共用專案而不是刪除專案，也可能會發生這種情況。

_首次回報於2024年4月25日。_

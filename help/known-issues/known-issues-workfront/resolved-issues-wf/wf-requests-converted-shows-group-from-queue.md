# 請求：轉換為具有範本之專案的請求會顯示來自請求佇列的群組，而非來自範本的群組

當使用者使用範本將請求轉換為專案時，新建立的專案會與擁有請求佇列的群組相關聯，而非範本上指派的群組。 即使在建立專案時，與範本相關聯的群組會填入 [!UICONTROL 群組] 欄位。

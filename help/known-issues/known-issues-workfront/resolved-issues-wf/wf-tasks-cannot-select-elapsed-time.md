# 任務：內聯編輯任務持續時間時無法選擇「已用時間」

>[!NOTE]
>
>此問題已於2022年7月21日修正。

當用戶正在查看任務清單並嘗試編輯 [!UICONTROL 任務持續時間]，則無法使用下列持續時間單位：

* [!UICONTROL 經過的分鐘數]
* [!UICONTROL 經過的時數]
* [!UICONTROL 經過的天數]
* [!UICONTROL 經過的週數]
* [!UICONTROL 經過的月數]

_2022年7月1日首次報告。_

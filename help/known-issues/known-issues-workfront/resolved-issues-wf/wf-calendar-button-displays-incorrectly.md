# 日曆：將項目新增至日曆時，按鈕顯示不正確

>[!NOTE]
>
>此問題已於2022年7月25日修正。

當使用者將項目新增至日曆並選取 [!UICONTROL 新增進階項目] 選項 [!UICONTROL 添加任務], [!UICONTROL 新增專案]，或 [!UICONTROL 新增元素] 按鈕與文本重疊。 按鈕仍可運作，使用者可以新增項目。

_2022年6月2日首次報導。_

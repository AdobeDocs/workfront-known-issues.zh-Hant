# [!UICONTROL 設定]:無法建立新狀態

>[!NOTE]
>
>此問題已於2022年8月1日修正。

當使用者嘗試建立新狀態時，狀態不會顯示在 [!UICONTROL 設定]. 當新狀態與已刪除狀態具有相同的索引鍵時，就會發生此情況。

_2022年7月12日首次報告。_

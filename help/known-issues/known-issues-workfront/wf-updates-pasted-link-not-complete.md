# 更新：複製的連結未正確貼上

如果使用者透過按一下滑鼠右鍵並選取「[!UICONTROL 複製連結地址]&quot;，然後貼上更新中的連結，該連結無法正確貼上。 只有連結的第一個部分是連結，其餘的URL則會被忽略。

使用&#39;&#39;以外的方法複製連結[!UICONTROL 複製連結地址]」可讓連結如預期貼上。

**解決方法**

使用&quot;[!UICONTROL 複製]」選項而非「[!UICONTROL 複製連結地址]「。

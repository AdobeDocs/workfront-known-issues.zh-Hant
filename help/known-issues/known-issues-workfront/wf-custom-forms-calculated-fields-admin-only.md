# 自訂表單：計算欄位未顯示非管理員的資料

當不是Workfront管理員的使用者檢視自訂表單時，其中欄位從表單僅限管理員區段中的資料提取，該欄位不顯示計算值。 如果使用者執行重新計算運算式，則會顯示值，但在重新整理時遺失。

Workfront管理員可如預期看到欄位值。

# 指派：任務指派失敗且繼承的許可權已移除

將任務指派給團隊時，指派失敗並傳回錯誤。 此外，繼承的許可權會被移除，使用者無法存取其嘗試指派的任務。

_於2024年9月5日首次通報。_

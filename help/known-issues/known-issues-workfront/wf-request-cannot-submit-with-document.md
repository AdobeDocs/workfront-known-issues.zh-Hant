# 請求：填寫請求時無法上傳檔案

>[!NOTE]
>
>此問題已在 2023 年 2 月 23 日修正。

當使用者填寫請求並嘗試上傳檔案時，檔案未上傳，且使用者會看到以下錯誤：

「上載錯誤：請嘗試較小的檔案，或在建立後附加檔案。」

_於2026年2月6日首次通報。_

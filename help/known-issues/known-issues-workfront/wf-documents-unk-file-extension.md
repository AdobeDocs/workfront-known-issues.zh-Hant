# 文件：連結來自整合的文件時出現 UNK 副檔名

<!--WF and WFP-->

>[!NOTE]
>
>此問題已關閉，因為這不是 Workfront 的問題。

當使用者從文件整合連結文件時，該文件將顯示在 Workfront 中，並帶有 UNK 副檔名。

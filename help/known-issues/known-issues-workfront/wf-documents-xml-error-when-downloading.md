# 文件：下載時發生 XML 錯誤

<!--WF, WFP TOCs-->

>[!NOTE]
>
>此問題已在 2023 年 7 月 21 日修正。

當使用者嘗試下載文件時，該文件未下載，使用者看到頁面出現，其中包含以下訊息，後面接著 XML 文字。

「[!UICONTROL  XML 檔案似乎沒有任何關聯的樣式資訊。文件樹如下。]」

超過 25 MB 的檔案有回報此問題。

_於 2023 年 7 月 20 日首次通報。_

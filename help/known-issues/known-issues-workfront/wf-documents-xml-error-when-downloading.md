# 檔案：下載時出現XML錯誤

<!--WF, WFP TOCs-->

當使用者嘗試下載檔案時，檔案未下載，且使用者會看到一個頁面，其中包含以下訊息，然後是XML文字。

&quot;[!UICONTROL XML檔案似乎沒有任何相關聯的樣式資訊。 檔案樹狀結構如下。]&quot;

已報告檔案超過25 MB的情況。

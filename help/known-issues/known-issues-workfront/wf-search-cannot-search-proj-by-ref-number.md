# 搜尋：無法按引用編號搜索項目

>[!NOTE]
>
>此問題已於2022年7月1日修正。

當使用者嘗試使用其參考編號來搜尋現有專案時，搜尋不會傳回專案。

_2022年6月27日首次報導。_

# Workfront規劃：無法將欄位新增至檢視

>[!NOTE]
>
>* 此問題已在2025年12月19日修正。
>* 此問題已在「自訂重新整理沙箱」中報告。

當使用者嘗試將欄位新增到Workfront規劃中的檢視時，未新增欄位，且使用者看到以下訊息：

「無法建立欄位欄。 再試一次。」

_於2025年12月10日首次通報。_

# [!UICONTROL 設定]：稽核記錄未匯出

當使用者嘗試匯出稽核記錄時，稽核記錄未匯出。 [!DNL Workfront] 中沒有稽核記錄未匯出的跡象，但預期的匯出檔案從未出現。

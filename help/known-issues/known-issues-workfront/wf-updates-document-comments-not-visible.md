# 更新：檔案註釋未出現在父物件的更新中

<!--WF, WFP-->

當使用者在檔案上進行註解時，該註解不會立即出現在檔案父物件的「更新」區域中。

此問題已在新評論體驗中報告。 註解會如預期出現在舊版註解體驗中。

_首次回報於2024年3月6日。_

# 更新：在摘要面板中使用@標籤使用者無效

當使用者嘗試透過在摘要面板的更新區域使用@標籤另一個使用者時，按一下下拉清單中的使用者名稱無效。 第二次嘗試標籤使用者如預期般運作。

**解決方法**

再次標籤使用者。 第二次嘗試如預期運作。

_於 2024 年 3 月 18 日首次通報。_

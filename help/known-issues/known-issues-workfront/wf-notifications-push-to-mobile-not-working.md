# 通知：推播通知至Workfront Mobile應用程式無法運作

當發生應觸發推播通知至Workfront Mobile應用程式的事件時，未收到通知。

即使已在電子郵件或應用程式內通知中收到通知，也可能會發生這種情況。

_於2024年12月17日首次通報。_

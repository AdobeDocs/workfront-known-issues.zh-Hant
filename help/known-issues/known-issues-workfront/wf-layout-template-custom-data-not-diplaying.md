# 版面配置範本：透過版面配置範本新增到任務摘要時未顯示自訂資料欄位

當管理員透過版面配置範本將自訂資料欄位新增到任務摘要區段時，對於檢視任務摘要區段的使用者，該欄位顯示為空白。

_於2024年10月2日首次通報。_

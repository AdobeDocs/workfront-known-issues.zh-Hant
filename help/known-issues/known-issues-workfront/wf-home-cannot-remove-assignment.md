# [!UICONTROL 首頁]:無法刪除 [!UICONTROL 首頁]

當使用者檢視 [!UICONTROL 家庭工作清單] 並嘗試刪除分配，頁面將刷新，且分配未被刪除。

**因應措施**

開啟問題，並從問題本身中移除指派。

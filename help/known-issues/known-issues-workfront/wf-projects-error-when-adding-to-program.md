# 專案：將專案新增到計畫時發生錯誤

>[!NOTE]
>
>此問題已在2025年8月21日修正。

當使用者嘗試將現有專案新增到計畫時，專案未新增，並且使用者會看到以下錯誤訊息：

「載入次要內容時發生錯誤……」

_於2025年7月4日首次通報。_

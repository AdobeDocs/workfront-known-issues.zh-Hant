# 任務：在一或多個任務中取消指派使用者

系統可能會自動將使用者從指派給他們的任務中取消指派。 在一或多個任務中可能會發生這種情況。 取消指派不會顯示在任務的系統更新區域中，儘管它會在設定選單的更新摘要區段中顯示。

_於2023年12月6日首次通報。_

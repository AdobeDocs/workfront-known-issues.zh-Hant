# [!UICONTROL 首頁]： [!UICONTROL 工作區] 和圖釘未載入

當使用者登入時，可能會發生以下情況：

* 使用者的新增 [!UICONTROL 首頁工作區] 未載入，且他們看到錯誤&quot;[!UICONTROL 我們無法載入您的工作區資訊。 請聯絡 Workfront，以便我們找出問題並加以修正。]」
* 使用者的釘選未載入，且他們看到錯誤&quot;[!UICONTROL 由於系統錯誤，您的釘選無法使用。 請嘗試重新整理瀏覽器以修正問題。]&quot;

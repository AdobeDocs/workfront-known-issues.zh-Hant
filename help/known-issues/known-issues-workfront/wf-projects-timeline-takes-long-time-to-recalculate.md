# 專案：變更任務日期時，時間表重新計算的速度非常慢

當使用者檢視專案並變更專案任務的日期時，會重新計算專案中的所有日期。 此程式可能需要幾分鐘的時間，在此期間，專案任務上的日期顯示為灰色並且無回應。

_首次回報於2024年2月8日。_

# 檔案整合：將資產拖曳至Experience Manager Assets整合中的資料夾時發生錯誤

當使用者嘗試將資產拖曳至透過Experience Manager Assets整合連結的資料夾時，檔案未移動，且使用者看到以下錯誤：

&quot;？document.action.upload.drag.noaccess？&quot;

**因應措施**

使用「新增」按鈕將資產新增至資料夾。

_於 2023 年 9 月 18 日首次回報。_

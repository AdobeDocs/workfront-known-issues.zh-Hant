# 校訂：從連結文件建立的校訂並未套用校訂範本

<!--On WF, WFF, WFP TOCs-->

<!--

>[!NOTE]
>
>This issue was fixed on March 14, 2024.

-->

當使用者從連結文件建立校訂時，校訂範本未正確套用，且校訂可能會遺失工作流程等資訊。

這也適用於透過 API 和 Workfront Fusion 建立的校訂。

_於 2024 年 2 月 28 日首次通報。_

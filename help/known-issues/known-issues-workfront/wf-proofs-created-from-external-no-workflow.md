# 校訂：從連結檔案建立的校訂未套用校訂範本

<!--On WF, WFF, WFP TOCs-->

當使用者從連結的檔案建立校訂時，校訂範本未正確套用，並且校訂可能缺少工作流程等資訊。

這也適用於透過API和Workfront Fusion建立的校樣。

_首次回報於2024年2月28日。_

# 校訂：校訂角色變更為「[!UICONTROL 唯讀]」

>[!NOTE]
>
>此問題已在 2023 年 3 月 16 日修正。

具有「[!UICONTROL 唯讀]」以外之校訂角色的使用者自動變更為「[!UICONTROL 唯讀]」。 沒有顯示此變更的通知或記錄。

_於 2023 年 2 月 8 日首次通報。_

# 請求：請求佇列不可用

>[!NOTE]
>
>此問題已在2025年3月14日修正。

當使用者嘗試選取特定請求佇列時，他們會看到訊息「請求佇列不再可用。」 即使請求佇列仍然存在，也可能會發生這種情況。

_於2025年3月13日首次通報。_

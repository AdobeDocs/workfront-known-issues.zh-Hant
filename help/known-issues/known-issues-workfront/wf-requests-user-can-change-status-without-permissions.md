# 請求：使用者即便在缺乏權限的情況下仍能變更狀態

>[!NOTE]
>
>此問題已結案，因為目前已如預期運作。

擁有投稿存取權，但沒有權限變更問題狀態的使用者，依舊可以變更狀態。

_於 2024 年 7 月 29 日首次回報。_

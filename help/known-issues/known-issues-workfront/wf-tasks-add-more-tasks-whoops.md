# 任務：新增個人任務時出現糟糕錯誤

當使用者在任務清單上的個人任務嘗試透過按一下「新增更多工」按鈕來建立新任務時，他們無法建立任務，並且會看到以下訊息：

「[!UICONTROL 糟糕！發生錯誤。請聯絡 Workfront，以便我們找出問題並加以修正。]」

**解決方法**

使用「新增任務」按鈕來建立任務。

# [!UICONTROL 更新]:無法記錄 [!UICONTROL 更新] 區域

當使用者嘗試在 [!UICONTROL 更新] 區域中，當天將顯示為灰色，用戶無法輸入任何小時。

**因應措施**

清除瀏覽器的快取或使用無痕視窗可暫時解決問題。

_於 2023 年 3 月 7 日首次通報。_

# 無法建立檔案清單的檢視

當使用者嘗試在檔案清單中建立新檢視時，畫面變為空白，使用者無法建立檢視。

現有檢視如預期般運作。

**因應措施**

管理員可以在「設定>介面>檢視」區域中新增檔案檢視。

_於2024年7月9日首次通報。_

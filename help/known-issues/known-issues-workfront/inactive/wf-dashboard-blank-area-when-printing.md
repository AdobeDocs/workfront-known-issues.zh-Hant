# 儀表板：將儀表板列印為 PDF 時空白邊界很大

<!--Article by request-->

當使用者以橫向模式將儀表板列印為 PDF 時，PDF 右側的空白邊界很大。

**解決方法**

以直向模式列印可顯示正確的邊界。

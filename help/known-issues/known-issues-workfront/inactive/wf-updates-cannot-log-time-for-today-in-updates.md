# [!UICONTROL 更新]：無法在[!UICONTROL 更新]區域記錄當天的時間

>[!NOTE]
>
>由於無法重現此問題，因此已結案。

當使用者嘗試在[!UICONTROL 更新]區域記錄當天的時間時，當天呈現灰色，使用者無法輸入任何時數。

**解決方法**

清除瀏覽器的快取或使用無痕視窗可以暫時解決問題。

_於 2023 年 3 月 7 日首次通報。_

# 行事曆：任務未顯示

>[!NOTE]
>
>產品團隊目前正在評估此問題的解決方案。解決此問題後，將在產品公告中進行溝通，而不是在維護更新中進行溝通。

當使用者擁有具有相同篩選器的清單報告和行事曆報告時，他們會看到某些任務顯示在清單報告中，但未顯示在行事曆報告中。

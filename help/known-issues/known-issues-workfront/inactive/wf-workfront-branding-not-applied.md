# [!DNL Workfront]：無法使用品牌化

>[!NOTE]
>
>此問題已結案，因為這不是問題。

>[!IMPORTANT]
>
>如果您的組織已遷移至 Adobe Admin Console、Adobe Unified Shell 或 Adobe IMS (身分管理系統)，則無法使用品牌化。

品牌化無法提供給已遷移至 Adobe Admin Console、Adobe Unified Shell 或 Adobe IMS (身分管理系統) 的組織。 當嘗試使用品牌化時，這些組織中的使用者可能會看到以下其中一個訊息：

* 品牌化無法在版面配置範本中作為設定。
* 當使用者將檔案上傳到設定的[!UICONTROL 品牌]區域時，檔案看起來已正確上傳。 但如果使用者查看[!UICONTROL 主選單]，他們沒有看到該標誌。 而是看到以下訊息：

  「[!UICONTROL 公司標誌無法使用。 我們無法根據提供的 URL 載入您的公司標誌。 請聯絡您的系統管理員。]」

_於 2023 年 9 月 29 日首次回報。_

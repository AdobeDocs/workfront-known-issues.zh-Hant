# 自訂表單：選取多個值時下拉式清單關閉

當使用者嘗試在自訂表單欄位上選擇多個值時，下拉清單會在選擇第一個值後關閉。

當欄位與自訂表單中的顯示邏輯相關聯時，就會發生這種情況。

_於2024年7月9日首次通報。_

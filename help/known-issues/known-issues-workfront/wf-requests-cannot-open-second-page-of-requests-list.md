# 請求：無法開啟請求清單的第一頁以外的頁面

>[!NOTE]
>
> 此問題已關閉，因為其可如預期運作。

當使用者檢視的請求清單有2000多個請求（因此會有多個頁面），並嘗試檢視清單的其他頁面時，「[!UICONTROL 頁面]「 」按鈕沒有響應，用戶無法查看清單的其他頁。

_2022年7月28日首次報告。_

# 任務/問題：獲指派的使用者未獲得物件存取權

>[!NOTE]
>
>這個問題已解決。

當使用者被指派至工作物件時，該使用者可能未獲得物件存取權，並且無法完成指派。該使用者可以請求存取權，但這會增加完成工作項目的時間和複雜性。

**因應措施**

執行下列任一項作業：

* 取消指派並重新指派使用者
* 手動為使用者提供存取權

_於 2023 年 7 月首次通報。_

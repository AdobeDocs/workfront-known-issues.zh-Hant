# 首頁：當使用者移動首頁Widget中的欄時建立的&quot;widget.list.view&quot;

當管理員在設定區域中檢視檢視清單時，他們會看到許多稱為「widget.list.view」的檢視。 使用者重新排列首頁Widget中的欄時，就會建立這些檢視。

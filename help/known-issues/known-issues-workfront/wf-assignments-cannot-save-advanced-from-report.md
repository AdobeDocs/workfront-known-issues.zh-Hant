# 指派：無法從報表儲存進階指派變更

當使用者嘗試從報告進行進階指派變更時，變更不會儲存，且使用者會看到以下訊息：

「422： APIModel INTERNAL不支援欄位removeOldAssignments」

**解決方法**

您可以從專案的工作清單變更進階工作指派。

_於2026年2月4日首次通報。_

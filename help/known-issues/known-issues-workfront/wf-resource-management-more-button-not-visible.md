# 資源規劃： [!UICONTROL 更多] 按鈕不可見

當使用者正在檢視資源規劃工具並嘗試按一下 [!UICONTROL 更多] 按鈕，按鈕不可見。

**解決方法**

按一下 [!UICONTROL 更多] 應為按鈕。 按鈕不可見，但可按一下。

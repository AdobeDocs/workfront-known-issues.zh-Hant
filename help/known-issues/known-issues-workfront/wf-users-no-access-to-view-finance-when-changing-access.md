# 使用者：無法更新使用者存取權

當使用者嘗試將另一個使用者的存取權從檢視更新為管理專案時，使用者無法更新存取權，他們看到以下訊息：

「您沒有足夠的存取許可權來檢視此專案的財務。」

_於2025年12月11日首次通報。_

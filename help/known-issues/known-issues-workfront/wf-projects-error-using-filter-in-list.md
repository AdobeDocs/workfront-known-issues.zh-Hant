# 專案：在產品清單上使用篩選器時發生錯誤

如果使用者正在檢視專案清單並套用篩選器，清單會消失且使用者會看到以下訊息：

&quot;[!UICONTROL 發生錯誤，我們正在努力解決此請求。 若要繼續您的工作，請嘗試重新整理此瀏覽器頁面。]」

使用新的篩選器體驗時已報告此問題。

_首次回報於2024年2月22日。_

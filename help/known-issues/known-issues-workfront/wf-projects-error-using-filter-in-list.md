# 專案：在產品清單上使用篩選器時發生錯誤

>[!NOTE]
>
>此問題已在2024年2月23日修正。

如果使用者正在查看專案清單並套用篩選器，該清單就會消失，且使用者看到下列錯誤訊息：

「[!UICONTROL 發生錯誤，我們正在努力解決這項要求。若要繼續您的工作，請嘗試重新整理此瀏覽器頁面。]」

使用新版篩選器體驗時已通報這項錯誤。

_於 2024 年 2 月 22 日首次通報。_

# 工時單：&quot;[!UICONTROL 未保存您最近的更改]&quot;在時間表上自動保存更改時出錯

>[!NOTE]
>
>此問題已於2022年10月6日修正。

當用戶嘗試以觸發自動保存更改的方式編輯時間表時，不會保存更改，用戶將看到以下消息：

「Y[!UICONTROL 未保存我們最近的更改。 重新整理頁面以檢視。]&quot;

編輯下列項目時，會回報此問題：

* 小時
* 任務

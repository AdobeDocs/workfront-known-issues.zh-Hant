# 登入身份：顯示管理員的首頁配置範本

當管理員以其他使用者身份登入並導覽到設定中的版面配置範本時，會顯示他們自己的首頁版面配置範本，而不是他們以登入的使用者身份顯示。

_於2025年1月10日首次通報。_

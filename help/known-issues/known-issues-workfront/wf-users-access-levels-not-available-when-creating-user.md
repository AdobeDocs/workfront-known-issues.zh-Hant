# 使用者：建立使用者時無法提供較低的存取層級

當使用者正在建立另一個使用者時，新建立的使用者只能使用第一個使用者的存取層級。 所有許可權低於建立使用者許可權的存取層級，都應可指派給新使用者。

_於2024年1月30日首次通報。_

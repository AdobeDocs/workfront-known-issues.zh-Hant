# 使用者無法選取「[!UICONTROL 預設時數類型]」或「[!UICONTROL 可用時數類型]」

當管理員正在編輯使用者並嘗試選取「[!UICONTROL 預設時數類型]」或「[!UICONTROL 可用時數類型]」時，他們看到這些欄位的下拉選單已停用，而且無法選取時數類型。

_於 2023 年 3 月 30 日首次通報。_

# 任務和問題：建立任務或問題時無意中退出

>[!NOTE]
>
> 此問題已於2022年9月8日修正。

當使用者在專案中建立任務或問題，並在建立快顯視窗外按一下時，快顯視窗會關閉，而且所有先前輸入的資訊都會遺失。

_2022年3月25日首次報導。_

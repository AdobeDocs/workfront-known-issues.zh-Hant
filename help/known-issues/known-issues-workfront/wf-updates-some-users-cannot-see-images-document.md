# 更新：有些使用者看不到附加至評論的影像或檔案

當使用者正在檢視物件的更新，並且更新已附加影像或檔案時，該影像或檔案未顯示在使用者的更新中。

已針對非Workfront管理員的使用者報告此問題。

_於2025年3月3日首次通報。_

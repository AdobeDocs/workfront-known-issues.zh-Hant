# 核准：我的核准Widget顯示不在使用者指派至的階段的校訂

當使用者在首頁檢視他們的我的核准Widget時，他們會看到將使用者指派到階段的校訂，但校訂不在指派使用者的階段。 這可能會讓使用者難以快速瞭解所需的動作。

# 自訂表單：自訂表單上顯示型別轉換錯誤

RTF欄位造成自訂表單中出現下列錯誤：

「錯誤：不允許在RTF文字間進行顯示型別轉換。」

這可能發生在以下情況中：

* 使用者開始編輯表單，但不做任何變更就按一下套用。
* 使用者正在建立自訂表單。

在這兩種情況下，RTF欄位都會造成問題。

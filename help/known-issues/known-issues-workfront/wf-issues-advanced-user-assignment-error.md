# 請求和問題：進階使用者指派錯誤

當使用者嘗試在請求或問題上進行進階使用者指派時，使用者未被指派，並且使用者會看到以下錯誤：

「發生錯誤，我們正在努力解決這個問題。若要繼續工作，請重新整理此瀏覽器頁面。」

重新整理瀏覽器頁面無效。

_於2024年12月26日首次通報。_

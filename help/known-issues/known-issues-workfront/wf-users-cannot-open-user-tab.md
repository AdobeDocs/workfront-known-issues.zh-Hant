# 使用者：開啟使用者頁面時發生錯誤

當使用者嘗試從主要選單開啟使用者頁面時，頁面未開啟，且使用者看到以下錯誤訊息：

「發生錯誤，我們正在努力解決這個問題。若要繼續工作，請重新整理此瀏覽器頁面。」

_於2025年12月8日首次通報。_

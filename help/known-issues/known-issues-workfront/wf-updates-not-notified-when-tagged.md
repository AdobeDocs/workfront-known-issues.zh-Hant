# 更新：標籤使用者無效

當使用者在評論中被標籤時，被標籤的使用者看不到該評論，此外，被標籤的使用者也不會收到有關評論的通知，無論是通過電子郵件還是透過應用程式內通知。

這已在舊版評論體驗中報告。

_於2024年3月1日首次通報。_

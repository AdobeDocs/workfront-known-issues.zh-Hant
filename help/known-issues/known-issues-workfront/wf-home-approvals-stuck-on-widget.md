# 首頁：核准卡在 [!UICONTROL 等待我的核准] Widget

<!--on WF and WFP TOCs-->

當上傳檔案的新版本且先前版本具有未完成的核準時，檔案的舊版本會卡在核准者的 [!UICONTROL 等待我的核准] Widget. 由於有新版本，因此無法核准核准，且無法從Widget中移除核准。

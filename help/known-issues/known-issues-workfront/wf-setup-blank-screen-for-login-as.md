# 設定：「登入身份」導向到空白畫面

>[!NOTE]
>
>此問題已在2024年8月8日修正。

當管理員以其他使用者身份登入時，管理員不會看到該使用者的帳戶，而是會看到一個空白畫面。

**解決方法**

重新整理畫面。 這可能需要數次嘗試。

_首次回報於2024年7月17日。_

# 控制面板：在控制面板中檢視的報表圖表使用與在報表中檢視不同的色彩

>[!NOTE]
>
>產品團隊目前正在評估此問題解決方案，這可能需要產品增強功能。 產品增強功能會在產品公告中傳達，而不會在維護更新中傳達。

使用者在控制面板中檢視報表圖表時，使用的顏色與使用者檢視報表本身時的顏色不同。 如果使用者預期特定顏色具有指定意義，則可能會造成混淆。

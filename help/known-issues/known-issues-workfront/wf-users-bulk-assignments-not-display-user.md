# 使用者：進行大量指派時遺失使用者

當使用者嘗試進行大量指派時，並且工作角色被指派給任務，具有該工作角色的使用者未出現在清單中，並且指派使用者無法選擇他們。

此行為也已在工作負載平衡器中報告。

_首次回報於2024年5月17日。_

# 儀表板：自訂儀表板未開啟

>[!NOTE]
>
>此問題已在2025年4月10日修正。

當使用者檢視物件並嘗試開啟附加到物件的自訂儀表板時，儀表板未開啟。

_於2025年2月13日首次通報。_

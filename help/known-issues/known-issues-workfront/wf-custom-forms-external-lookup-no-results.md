# 自訂表單：外部查詢欄位未傳回結果

當外部查詢欄位參考僅選取一個值的多選欄位時，該欄位未傳回值。

例如，如果外部查詢欄位參考同時選取「紅色」和「藍色」值的多重選取欄位，則該欄位會如預期運作。 如果欄位僅選取「紅色」，則外部查詢欄位不會傳回任何值。

_首次回報於2024年4月2日。_

# 用戶：用戶無法編輯自己的通知設定

若使用者具有 [!UICONTROL 工作人員] 許可證會嘗試編輯自己的通知設定， [!UICONTROL 通知] 選項不會顯示在 [!UICONTROL 編輯] ，而使用者無法編輯設定。

_首次回報於 2023 年 2 月 9 日。_

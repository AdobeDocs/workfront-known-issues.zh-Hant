# 校訂：無法關閉校訂檢視器

<!--
>[!NOTE]
>
>This issue was fixed on October 24, 2024.
-->

當使用者在校訂檢視器中檢視校訂，並嘗試按一下右上角 X 關閉校訂時，校訂不會關閉。

**解決方法**

嘗試以下任一作業：

* 透過檔案清單存取校訂。 如此可能會讓 X 照常運作。
* 再次嘗試關閉校訂。
* 透過關閉瀏覽器標籤來關閉校訂。

_於2024年11月5日首次通報。_

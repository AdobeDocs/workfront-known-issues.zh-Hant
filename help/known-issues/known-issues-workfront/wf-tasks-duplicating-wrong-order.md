# 任務：複製任務導致任務顯示順序錯誤

當使用者在任務清單中選擇一個或多個任務並複製它們時，任務清單然後根據任務順序編號以錯誤的順序顯示任務。 任務編號正確，但顯示順序錯誤。

**因應措施**

重新整理該頁面。

# 通知：文件核准 Beta 版的通知其格式不正確

當使用者收到來自新文件核准 Beta 版的電子郵件時，該電子郵件的格式不正確。

適用於 Windows 的 Microsoft Outlook 中已回報此問題。

_於 2024 年 3 月 14 日首次通報。_

# [!UICONTROL 工作負載平衡器]:小時數不會調整為因用戶休假而引起

如果用戶計畫超時並分配了任務，則工作負載平衡器不會更新時間軸以說明超時。 反之， [!UICONTROL 工作負載平衡器] 顯示使用者在已排程的天數中被過度分配。 要調整工時，管理員必須進入項目並重新計算時間軸。

_2023年5月6日首次報導。_

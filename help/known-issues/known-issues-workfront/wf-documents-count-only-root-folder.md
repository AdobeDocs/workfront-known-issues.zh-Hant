# 檔案：檔案計數不準確

當使用者檢視附加了檔案的物件時，左側導覽中的檔案索引標籤可能會顯示不準確的檔案數量。 顯示的數字是根資料夾中的檔案數。 不會計運算元資料夾中的任何檔案。

_於 2024 年 6 月 20 日首次通報。_

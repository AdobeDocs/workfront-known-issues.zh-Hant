# [!DNL Workfront Proof]:新增至校樣的使用者只有「檢視」存取權

>[!NOTE]
>
>此問題已於2022年7月25日修正。

將使用者新增至校樣時，他們只會收到「[!UICONTROL 檢視]」查閱該校樣。 已透過下列角色回報此問題：

* [!UICONTROL 仲裁者]
* [!UICONTROL 檢閱者]
* [!UICONTROL 核准者]

_2022年6月1日首次報導。_

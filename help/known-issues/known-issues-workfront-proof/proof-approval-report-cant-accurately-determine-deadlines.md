# 校樣： [!UICONTROL 證明核准] 報告無法準確確定截止時間

注意：產品團隊目前正在評估此問題解決方案，這可能需要產品增強功能。 產品增強功能會在產品公告中傳達，而不會在維護更新中傳達。

當使用者建立 [!UICONTROL 證明核准] 包含證明截止日期的報表、報表中顯示的日期不正確，進而導致報表誤算和結果中的錯誤資訊。

2020年12月21日首次報告

# [!DNL Workfront Proof]:資料夾檢視未顯示校樣

>[!NOTE]
>
>此問題已於2022年7月28日修正。

當使用者將校樣移至資料夾時，會有無法判斷的延遲，校樣不會出現在資料夾樹狀結構中。

_2022年6月7日首次報導。_

# 校樣：在中顯示的校樣批准截止日期不正確 [!UICONTROL 家庭工作清單]

>[!NOTE]
>
>此問題已於2022年9月1日修正。

將使用者指派給校樣核準時，該校樣核准會顯示在使用者的 [!UICONTROL 家庭工作清單] 具有未來最遠的階段的截止期，而無論該用戶分配給哪個階段。

換言之，指定校樣的所有階段的所有批准者在其中看到相同的截止日期 [!UICONTROL 家庭工作清單]. 這個最後期限是未來最遠的舞台的最後期限。

2022年4月11日首次報導。

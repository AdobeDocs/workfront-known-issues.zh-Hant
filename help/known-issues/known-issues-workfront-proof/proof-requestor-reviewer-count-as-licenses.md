# [!DNL Workfront Proof]:請求者和審核者會計入組織的授權計數

當使用者以「審核者」或「請求者」的身分新增至校樣時，會獲得「[!UICONTROL 訪客]&quot;權限設定檔，不應使用 [!DNL Workfront Proof] 授權。 不過，新增使用者時，使用的計數 [!DNL Workfront Proof] 許可證數量增加。

_2022年6月23日首次報導。_

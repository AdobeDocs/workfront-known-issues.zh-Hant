# [!DNL Workfront Proof]:在中初始配置後，無法修改用戶選項 [!UICONTROL 工作流程範本]

>[!NOTE]
>
>本文已於2022年8月18日修正。

當使用者新增使用者至 [!UICONTROL 工作流程範本]，則可設定選項。 不過，初始設定完成後，使用者無法再修改下列項目：

* &quot;[!UICONTROL 解決注釋並應用操作]「 」能力
* &quot;[!UICONTROL 透過標籤共用校樣]「 」能力
* 校樣角色([!UICONTROL 審核者], [!UICONTROL 核准者]等)

**2022年5月31日首次報導。**

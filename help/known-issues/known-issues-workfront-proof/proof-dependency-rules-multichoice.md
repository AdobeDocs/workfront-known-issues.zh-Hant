# [!DNL Workfront Proof]：整合帳戶 — 校樣相依性規則不允許選擇多個欄位

只能為內的相依性規則選取清單和選項按鈕自訂欄位型別 [!DNL Workfront] 和 [!DNL Workfront Proof] 整合的執行個體。 與ProofHQ舊帳戶相比，這是一項有限的功能。

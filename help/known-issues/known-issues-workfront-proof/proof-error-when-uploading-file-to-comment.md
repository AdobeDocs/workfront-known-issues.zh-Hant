# [!DNL Workfront Proof]:將檔案上傳到注釋時出錯

>[!NOTE]
>
>此問題已於2022年8月25日修正。

當使用者嘗試將檔案上傳至 [!DNL Workfront Proof]，則檔案不會上傳，且使用者會看見下列錯誤訊息：

[!UICONTROL 技術困難：無法上傳1個檔案。 請再試一次，或聯絡客戶支援以取得協助。]

_2022年8月4日首次報導。_

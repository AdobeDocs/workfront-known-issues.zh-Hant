# 校樣：使用者無法自行移至校樣的其他階段

>[!NOTE]
>
>此問題已於2022年6月24日修正。

當使用者檢視校樣的「校樣工作流程」並嘗試將自己拖曳至校樣的不同階段時，使用者的名稱會對齊回原始階段，而不會新增至所需階段。

2022年6月2日首次報導。

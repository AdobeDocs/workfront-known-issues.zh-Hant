# [!DNL Workfront Proof]:檢視含有多頁的校樣時出現空白頁面

當使用者檢視含有多個頁面的校樣時，使用者可以看到頁面的縮圖，但頁面沒有在主檢視器中開啟。

_2022年9月22日首次報導。_

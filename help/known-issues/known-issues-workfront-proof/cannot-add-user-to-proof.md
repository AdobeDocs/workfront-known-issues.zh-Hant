# [!DNL Workfront Proof]:無法通過在注釋中標籤用戶來向校樣添加用戶

>[!NOTE]
>
>產品團隊目前正在評估此問題解決方案，這可能需要產品增強功能。 產品增強功能會在產品公告中傳達，而不會在維護更新中傳達。

當使用者為留言中目前未在校樣上的其他使用者加上標籤時，他們無法為使用者選取工作角色或電子郵件，因此無法將使用者新增至校樣。

2022年3月21日首次報導。

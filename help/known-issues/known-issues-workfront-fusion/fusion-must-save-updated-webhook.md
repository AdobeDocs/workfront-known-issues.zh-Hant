# Workfront Fusion： webhook的更新未自動儲存

當使用者對webhook進行變更(例如在「Workfront >觀看事件」模組中)時，變更未自動儲存。

使用者&#x200B;**必須儲存情境**，才能保留對webhook所做的變更。

**解決方法**

儲存情境。

_首次回報於2025年7月17日。_

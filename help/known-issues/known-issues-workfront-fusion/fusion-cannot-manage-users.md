# Workfront Fusion：無法從Workfront Fusion新增或刪除使用者

當使用者嘗試在Workfront Fusion中新增或刪除使用者時，他們無法這麼做。

您可透過Adobe Admin Console新增或刪除使用者。 因此，此問題僅存在於尚未完全加入Adobe Admin Console的Fusion組織中。

**解決方法**

此問題是將組織移轉至Adobe Admin Console的程式中的暫時性副作用。 當您的組織完全移轉後，即可在Adobe Admin Console中管理使用者，不再需要在Workfront Fusion應用程式中管理使用者。

# Workfront Fusion：連線至沙箱環境時，Workfront事件訂閱版本2選項無法使用

>[!NOTE]
>
>沙箱環境已報告此問題。

當使用者設定監視Workfront沙箱環境中事件的Workfront模組時，無法使用基於Workfront事件訂閱服務版本2的條件。

_於2025年3月10日首次通報。_

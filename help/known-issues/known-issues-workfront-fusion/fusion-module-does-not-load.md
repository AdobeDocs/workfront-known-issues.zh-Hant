# [!DNL Workfront Fusion]:建立或編輯案例時模組未載入

當使用者建立或編輯案例並嘗試開啟模組時，模組不會開啟。 使用者看見旋轉的圓圈，指出模組正在載入，但模組未完成載入。

_2022年9月12日首次報導。_

# Workfront Fusion：從現在開始設定「從現在」處理所有資料

為案例選取「從現在開始」作為「選擇何時開始」設定後，如果使用者將案例回復到先前的版本，那麼它將嘗試處理所有資料，而不管日期如何，而不是僅包含選取開始選項後的資料。

_於2024年8月14日首次通報。_

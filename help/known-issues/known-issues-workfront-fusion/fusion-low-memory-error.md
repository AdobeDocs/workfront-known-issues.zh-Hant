# Workfront Fusion：記憶體不足錯誤

>[!NOTE]
>
>產品團隊目前正在評估此問題的解決方案，這可能需要產品增強功能。產品增強功能是在產品公告中而不是在維護更新中傳達。

情境執行失敗，出現記憶體不足錯誤。即使該情境之前執行成功，也可能會發生此狀況。

**解決方法**

將情境分解為幾個較小的情境。

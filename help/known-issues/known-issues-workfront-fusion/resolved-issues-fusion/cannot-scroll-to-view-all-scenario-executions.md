# [!UICONTROL Workfront融合]:無法滾動以查看所有方案執行

>[!NOTE]
>
>此問題已於2022年6月9日修正。

當使用者檢視案例的執行歷史記錄並嘗試向下捲動以檢視更多執行時，執行會停止載入，且使用者無法檢視執行。 此外，用戶無法向上滾動到最新的執行。

2022年5月18日首次報導。
